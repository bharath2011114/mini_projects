# Mini Projects (Login, Sign Up & Registration Forms)

## Description

This repository contains a set of mini web projects developed using HTML and CSS.
The project focuses on designing user interface forms such as Login Page, Sign Up Page, and Registration Form.

These pages demonstrate form design, layout structuring, and basic input validation using HTML.

---

## Objective

The main objectives of this project are:

* To design user interface forms using HTML and CSS
* To understand form elements such as input fields, labels, buttons, and checkboxes
* To create structured and user-friendly layouts
* To practice basic validation using HTML attributes

---

## Technologies Used

* HTML5
* CSS3

---

## Projects Included

### 1. Login Page

* Contains email and password fields
* Includes "Remember Me" option
* Provides links for "Forgot Password" and Sign Up
* Simple and clean centered layout

---

### 2. Sign Up Page

* Includes personal details such as name, email, phone number, and password
* Gender selection using radio buttons
* Workshop interests using checkboxes
* Uses fieldset and legend for better structure
* Basic validation using HTML attributes (required, pattern, minlength)

---

### 3. Registration Form

* Split layout design (Information section + Form section)
* Includes first name, last name, email, and password fields
* Terms and Conditions checkbox
* Styled using Flexbox for layout alignment
* More advanced UI compared to other forms

---

## Features

* Clean and simple user interface design
* Centered form layout using Flexbox
* Input validation using HTML attributes
* Hover effects on buttons
* Structured form elements for better usability

---

## How to Run the Project

1. Download or clone the repository
2. Open any HTML file (login.html, signup.html, registration.html) in a browser

---

## Learning Outcomes

* Understanding of HTML form elements
* Styling forms using CSS
* Using Flexbox for layout design
* Implementing input validation without JavaScript
* Designing user-friendly form interfaces

This project helped in understanding how to design and structure web forms using HTML and CSS.
